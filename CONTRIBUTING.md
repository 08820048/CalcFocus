# Contributing

Thanks for contributing to CalcFocus.

Repository: https://github.com/08820048/CalcFocus_Pro

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Create a feature branch.
4. Make your changes.
5. Run the relevant checks before opening a pull request.

## Local Workflow

```bash
npm install
npm run dev
```

## Before Opening a PR

- Keep changes scoped
- Test the behavior you touched
- Document any new setup or release requirements
